# Lab04: Unit Testing

## Overview

Console TicTacToe for two players.

## Getting started

* Clone the repo;
* Build with `dotnet build --configuration Release`;
* Find the dll in `bin/Release` folder;
* Start with `dotnet Lab04_TicTacToe.dll`;
